# 06 - Variaveis de Ambiente

## O que foi implementado

- dotenv carregado no backend.
- Validacao obrigatoria de variaveis criticas.
- .env.example criado sem dados reais.
- Validacao de formato e limites para variaveis numericas e origem CORS.

## Variaveis obrigatorias

- PORT
- DB_HOST
- DB_USER
- DB_PASSWORD
- DB_NAME
- JWT_SECRET
- JWT_EXPIRES_IN
- COMPANY_LATITUDE
- COMPANY_LONGITUDE
- ALLOWED_RADIUS_METERS
- CORS_ORIGIN

## Arquivos principais

- .env.example
- src/config/env.js
